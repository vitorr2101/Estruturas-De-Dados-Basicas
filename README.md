# Estruturas-De-Dados-Basicas
Alguns programas em C que implementam uma fila, uma pilha e uma árvore. Objetivando fornecer referências para uma aula introdutória de Estrutura de Dados, dentro de um contexto de gamificação, enquanto partes de uma 'busca ao tesouro'.

Separação de códigos por equipe:
1. Designados para a Fila de Sucessão, usem herdeiros.c
2. Designados para a Pilha de Segredos, usem diarios.c
3. Designados para a Árvore Genealógica, usem arvore.c
4. Designados para o Mapa da Mansão, usem mapa.jpg
